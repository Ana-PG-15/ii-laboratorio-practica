Preguntas Introductorias

¿Qué es un framework?

El framework es una plantilla o un esquema conceptual que nos ayuda en la elaboración de una tarea, ya que solo se completa de acuerdo a los que se quiere realizar.

¿Qué es Bootstrap 5 y cuál es su origen?

Bootstrap es un framework con código abierto para diferentes diseños y sitios web. Comenzó en Twitter para poder crear mejores herramientas internas, inició con HTML y CSS, llamándose así originalmente “ Bootstrap de Twitter.

¿Qué es diseño responsive en páginas web?

El diseño responsive es un estilo de página web que hace accesible y adaptable a todos tipo de dispositivo un sitio web.

¿Qué son los Github pages?

Es un sitio para poder alojar los nuevos proyectos en una página web, sin necesidad de conocer los servidores.

¿Cuál es la diferencia entre elemento y componente web?

El lenguaje de programa tiene elementos que se utilizan como bloques constructivos, mientras que los componentes web son bloques de código que encapsulan la estructura interna de elementos HTML, CSS y JavaScript.

¿Cuál es la importancia de documentar o comentar código?

La documentación es tan necesario como escribir el código, esto sirve para indicar porqué hace cada cosa y cambio, esto debe ser conciso y descriptivo a la vez.

¿Por qué existe la sección de atribución en un sitio web?

Estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios.

Integración de Conocimientos

HTML y CSS

El HTML es un lenguaje que permite definir la estructura del contenido de una página web a través del uso de elementos envueltos en etiquetas compatibles universalmente y el lenguaje CSS es el que describe cómo nuestros elementos en HTML serán mostrados en una pantalla de computadora, celular u otro dispositivo multimedia.

Maquetado Responsive

El maquetado responsive facilita la lectura y el acceso a los contenidos del portal o web, para hacer más fácil su uso. También facilita un acceso más rápido, ya que tardan menos en cargarse al estar optimizadas para estos dispositivos.

GitHub Pages

GitHub Pages es una forma de alojar nuestros proyectos en una página web, sin necesidad de tener conocimientos en servidores.

Bootstrap 5

Bootstrap es el framework más popular para el desarrollo de interfaces web con un diseño moderno, adaptadas a las pantallas de los dispositivos.
